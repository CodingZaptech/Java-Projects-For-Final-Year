# ATM Monitoring And Management System (Backend)

Here, the backend application code for managing ATMs in a system has been provided.

## Features
- Add new ATM
- View all ATMs
- Delete ATM

## Tech Stack
- Java 17
- Spring Boot 3
- Maven

## Run
mvn spring-boot: run

GET /atms – all ATMs

POST /atms – add ATM (JSON body)

DELETE /atms/{id} – delete ATM
