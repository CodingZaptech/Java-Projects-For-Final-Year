# Resume Building System (Backend)

The backend codes for creating and managing resumes in a system.

## Features
- Create new resume
- View all resumes
- Delete resume

## Tech Stack
- Java 17
- Spring Boot 3
- Maven

## Run
mvn spring-boot: run

GET /resumes – all resumes

POST /resumes – create resume (JSON body)

DELETE /resumes/{id} – delete resume
